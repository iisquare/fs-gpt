## fs-gpt

## 如何使用

### 环境安装

- 创建环境
```
conda create -n fs-gpt python=3.10
conda activate fs-gpt
```

- 安装项目
```
pip install -e ".[torch, embedding, inference]"
```

- 运行服务
```
fs-gpt run examples/embedding.yaml
```
